# Getting Started with PostgreSQL 🐘

Learning PostgreSQL is super fun — let's explore everything one by one!
A growing collection of SQL scripts documenting my hands-on journey with PostgreSQL.

## 📌 Topics Covered

- **Basics**: SELECT, DISTINCT, WHERE, ORDER BY, LIMIT, OFFSET
- **Tables**: Creating tables, Inserting values, Primary & Foreign Keys
- **Filtering**: LIKE, ILIKE, BETWEEN, IS NULL / IS NOT NULL, COALESCE
- **Joins**: Inner, Left, Right, Full, Self, Many-to-Many
- **Aggregation & Grouping**: Aggregate functions, combining tables
- **Window Functions**: PARTITION BY, LAG(), LEAD(), Window Frames
- **Advanced**: CTEs, Subqueries, Type Casting, Date & Time Arithmetic

## 🗂️ Structure

Each `.sql` file focuses on a specific concept with practice queries and examples.

## 🎯 Purpose

A personal practice space for mastering PostgreSQL — built one query at a time.
