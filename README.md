## 🎯 Purpose

This repo serves as my personal PostgreSQL learning log — each script focuses on a specific concept with worked examples, making it easy to revisit and reinforce topics as I go deeper into SQL and database fundamentals.

## 🛠 How to Use

1. Clone the repository
```bash
   git clone https://github.com/sajin-oops/Getting-started-with-postgreSQL.git
```
2. Open any `.sql` file in your favorite editor or PostgreSQL client (e.g., pgAdmin, DBeaver, psql).
3. Run the queries against a local PostgreSQL instance to see the concepts in action.
